# Neo Brutalist WordPress Theme

A high-contrast, bold WordPress theme designed for digital product downloads.

*   **Neo-Brutalist Design:** High contrast borders, bold typography, and vibrant "glitch" colors (Fuchsia, Cyan, Lime).
*   **Strict Architecture:** Built using PSR-12 coding standards, strict type hinting and a modular class-based structure.
*   **Tailwind CSS:** Fully styled using utility-first CSS for rapid and consistent UI development.
*   **Custom Download System:** Integrated metabox functionality to handle direct file downloads (PDF/Zip)
*   **Gallery Integration:** Transform standard WordPress galleries into interactive "Hero" image sliders.
*   **Accessibility Ready:** Semantic HTML5 markup, ARIA labels, skip-to-content links, and keyboard-navigable interfaces.
*   **Developer Friendly:** Clean separation of code (`inc/`) and views (`template-parts/`)
*   **Browser-Style Cards:** Unique card design with retro browser windows with dynamic "address bars" showing post slugs.
